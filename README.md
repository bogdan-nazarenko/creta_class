# 11ty + SCSS

## Template (custom)

1. **Install Node.js** if not already installed: [https://nodejs.org/](https://nodejs.org/)

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Start development:**

    ```bash
    npm start
    ```

4. **Format your source files:**

    ```bash
    npm run format
    ```

5. **Build for production (dist):**

    ```bash
    npm run build
    ```
